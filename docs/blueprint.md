# **App Name**: AI Job Portal

## Core Features:

- User Authentication: Secure user authentication with role-based access control, where the initial superadmin user has email hector.vargas@altiaweb.com.
- Candidate Profile Analysis: Analyze candidate profiles against job descriptions using AI to identify skill matches and gaps. Uses LLM to extract salient skills.
- Assessment Recommendation: Recommend suitable psychometric and technical evaluations for job roles based on job requirements. Uses LLM as a tool to find the best assessments.
- Interview Question Generation: Generate competency-based interview questions. It can be a mix of generic questions and individual questions personalized to each candidate. Uses LLM as a tool.
- Candidate Ranking: Rank candidates based on their profile analysis, assessment scores, and interview performance, using an AI algorithm. Uses LLM as a tool.
- Job Posting Management: Allow HR staff to create, edit, and manage job postings.
- Candidate Database: Store and manage candidate profiles, assessment results, and interview feedback in a database.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism, relevant to HR.
- Background color: Very light grey (#F5F5F5), providing a clean and neutral backdrop.
- Accent color: Soft green (#8BC34A), used sparingly to highlight key actions and positive feedback.
- Body and headline font: 'Inter' sans-serif for a modern, readable experience.
- Use professional, consistent icons for job roles, skills, and assessment types.
- Clean and structured layout for easy navigation and information accessibility.
- Subtle transitions and loading animations to provide a smooth user experience.
